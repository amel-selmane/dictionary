# English Dictionary application
**See the project** : amel-selmane.github.io/dictionnary/

## How it works

* To install the project : `npm i`
* To run the project : `npm run dev`
* To build the project : `npm run build`

## Technologies used

* Vite.js
* React.js
* Typescript
* TailwindCSS
* Vitest
